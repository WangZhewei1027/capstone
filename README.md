## 使用方法

### 安装

用的node v20

npm install安装所有包

### add.mjs

node add.mjs -h 来查看使用方法

手动按提示操作：node add.mjs

一次性输入参数：node add.mjs --workspace "10-04" --model "gpt-5-mini" --question "创建一个冒泡排序演示"

http://127.0.0.1:5500/viewer.html 可以快速可视化查看效果。但是workspace要手动添加

## 注意事项
