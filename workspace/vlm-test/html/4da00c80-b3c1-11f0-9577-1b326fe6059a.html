Binary Search

Binary search is an efficient algorithm for finding an item from a sorted list of items. It works by repeatedly dividing in half the portion of the list that could contain the item, until you've narrowed down the potential locations to just one.

How It Works

1. Begin with the middle element of the sorted array.
2. If the value of the middle element is less than the target value, narrow your search to the upper half of the array.
3. If the value of the middle element is greater than the target value, narrow your search to the lower half of the array.
4. Repeat the process until the target value is found or the subarray size becomes zero.

Pseudocode

function binarySearch(array, target):
    left = 0
    right = length(array) - 1

    while left <= right:
        mid = left + (right - left) / 2
        if array[mid] == target:
            return mid
        else if array[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

Time Complexity

The time complexity of binary search is O(log n), making it much more efficient than a linear search, especially for large datasets.
