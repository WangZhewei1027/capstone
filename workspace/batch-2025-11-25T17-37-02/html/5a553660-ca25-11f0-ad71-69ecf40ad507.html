Here is an example of a binary search tree (BST) using JavaScript: ``` class TreeNode { constructor(val, left = null, right = null) { this.val = val; this.left = left; this.right = right; } } function insert(root, val) { if (!root) return root; const temp = root; while (temp.left!== null && temp.val < val) { temp = temp.left; } if (temp === null) { temp = new TreeNode(val); } else { temp.left = insert(temp.left, val); } if (temp.right!== null) { temp = insert(temp.right, val); } root = temp; } ``` This function takes a `TreeNode` object as input and inserts it into a BST. The `insert()` function recursively searches for the minimum value to be inserted in the BST. It checks if the current node has a left child and if its value is less than the target value. If so, it returns the node. If not, it creates a new node with the target value and passes it to the left subtree. If the current node has a right child and its value is greater than the target value, it returns the node. If not, it recursively calls itself on the left subtree to find the smallest node that can contain the target value. To use this function, you would need to create a `TreeNode` object and pass it to the `insert()` function. For example: ``` const root = new TreeNode(1, null, null); root.insert(root, 3); console.log(root.val); // Output: 3 ```
