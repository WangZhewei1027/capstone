Hash Map Explorer — Separate Chaining

Interactive module to learn how a hash map maps keys to buckets and resolves collisions with separate chaining.
Learning Objective
After using this module you will be able to:
  • Compute a hash for a string key and map it to a bucket index.
  • Observe how collisions are handled using separate chaining (linked lists per bucket).
  • Trace insertion, lookup, and deletion through animated visual steps.
Interaction Design
Interact with controls to insert, lookup, or delete string keys. Actions animate a key chip moving into the calculated bucket. During lookup/delete the module sequentially highlights nodes in the bucket's chain to simulate comparisons. Controls:
  • Type a key and click Insert / Lookup / Delete.
  • Change table size or hash function to explore collision behavior.
  • Adjust speed or try Random key to see collisions emerge.
Layout Description
Two-column layout: controls & explanation on the left, visualization on the right. Spacing uses a 24px safe margin and at least 16px gap between interactive controls. The visualization shows a horizontal array of buckets with vertical chains. Accessible labels and keyboard focus are provided for major controls. The UI is responsive: on narrow screens it stacks vertically.
