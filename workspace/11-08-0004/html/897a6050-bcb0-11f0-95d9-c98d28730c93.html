Linear Search — Step-by-step Explorer

Explore how linear search scans an array to find a target value — step-by-step, with live feedback.
Learning objective
After using this module you will be able to:
  • Describe linear search behavior and worst-case comparisons.
  • Visually follow each comparison and recognize when the algorithm stops.
  • Experiment with array sizes, target placement, and speed to see effects on performance.
Interaction design
Interact using the controls below:
  • Enter a target value or pick one by clicking a cell.
  • Use Step to advance one comparison, Play to animate, Pause to stop, and Reset to start over.
  • Adjust array size and speed. Double-click a cell to edit its value. Controls provide keyboard focus and shortcuts (Space=Step, Enter=Play/Pause).

Explanation
Linear search checks each element in order until it finds the target or reaches the end. Worst-case it checks every element (O(n) comparisons). Try placing the target at different positions to observe.
Controls quick guide
