Adjacency List — Interactive Module

Visualize and manipulate a graph while watching its adjacency list update in real time.
Safe margin: 24px · Min control gap: 16px · Vanilla HTML/CSS/JS
Concept Title: Adjacency List (graph representation)
Learning Objective: After interacting, users will be able to recognize how an adjacency list captures a graph's connections, how updates (add/remove nodes or edges) affect the adjacency lists, and how directed vs. undirected edges differ in representation.
Interaction Design (click to expand)
- Add nodes by clicking "Add Node" or click on the canvas to place a node.
- Drag nodes to rearrange them; node positions update edges in real time.
- Add an edge using the source/target selectors; toggle directed/undirected.
- Click an edge to remove it; click a node to highlight and see its adjacency list.
- Visual feedback: edge-draw animation when created; nodes glow on select; adjacency list updates instantly with a readable array/object format. Keyboard: nodes are focusable and can be selected via Tab/Enter; arrow keys nudge selected node for accessibility.
