Hash Map — Visual Explorer (Separate Chaining)

Interactive module demonstrating how a hash map places keys into buckets and resolves collisions with linked lists (separate chaining).

Concept Title
Hash Map: bucket indexing and collision handling using separate chaining
Learning Objective
After interacting, you'll understand how keys are hashed to bucket indices, how collisions are stored as chains, and how search/insert/delete walk the chain.
Interaction Design
Type a key and value then Insert. Watch the computed hash and an animated bubble fly into the target bucket. If collisions occur, nodes stack as a linked list. Use Find/Delete to step through the chain visually. Adjust bucket count to see distribution change.
Layout Description
Left column holds controls and textual plan. Right column is the canvas showing buckets in a responsive grid. A floating animated bubble bridges controls and canvas when inserting to reinforce mapping. Safe margins of 24px, min spacing of 16px between interactive elements; responsive stacking on narrow screens for accessibility.
Change bucket count to see how modulo affects distribution (2–32).
