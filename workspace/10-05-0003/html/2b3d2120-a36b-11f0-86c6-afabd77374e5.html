Stable In-place Adaptive
Insertion sort builds a sorted region by inserting each element (the key) into its correct position. It performs well on small or nearly sorted arrays and requires no extra memory beyond a constant amount.
Time complexity: - Best: O(n) when nearly sorted (minimal shifts) - Average: O(n^2) - Worst: O(n^2) e.g., reversed input Space: O(1)
Status: Ready
Tip: Try "Nearly sorted" or "Reversed" to see best vs worst-case behavior.
