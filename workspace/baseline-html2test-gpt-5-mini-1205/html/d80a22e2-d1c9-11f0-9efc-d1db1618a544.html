Internals & Implementation
This demo uses an array-backed binary heap. For index i: - left child = 2*i + 1, right child = 2*i + 2 - parent = floor((i-1)/2)
Behavior: - Min-heap: parent <= children (smallest at root). - Max-heap: parent >= children (largest at root).
