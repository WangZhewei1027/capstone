Part 1: FSM-Based Evaluation Differentiates AI Model Quality

Research Question: Can FSM-based evaluation effectively differentiate the quality of interactive explanations generated by different AI models?

Methodology

  • Dataset: 219 HTML samples from 5 AI models
  • Evaluation Method: FSM Similarity (Structural 40% + Semantic 40% + Isomorphism 20%)
  • Analysis Approach:
    • Overall model comparison
    • Category-wise stratified analysis by CS concepts
    • ANOVA test for statistical significance

Statistical Significance: The p-value < 0.05 indicates that inter-model differences are statistically significant. The FSM-based evaluation method effectively differentiates the quality of HTML outputs across AI models.

Key Findings

Finding 1: Significant Model Performance Differences
The best performing model gpt-5-mini (32.10%) outperforms the lowest performing model Qwen1.5-0.5B-Chat (23.57%) by 8.53 percentage points.
Finding 2: FSM Evaluation Demonstrates Discriminatory Power
ANOVA F-statistic = 3.3670, p-value < 0.05. This indicates that the FSM-based evaluation method can significantly differentiate model quality.
Finding 3: Model Consistency Analysis
The standard deviation ranges from 5.09 to 18.77, indicating varying levels of output consistency across models.
