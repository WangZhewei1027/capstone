JavaScript Set Data Structure Demo

What is a Set?

A Set is a collection of unique values. Each value can occur only once in a Set.

Key features:

  • Stores only unique values (no duplicates)
  • Maintains insertion order
  • Provides efficient methods for adding, removing, and checking values

Basic Set Usage

// Creating a Set
let mySet = new Set();

// Adding values
mySet.add('apple');
mySet.add('banana');
mySet.add('apple'); // Duplicate - won't be added

// Checking size
console.log(mySet.size); // 2

// Checking if value exists
console.log(mySet.has('banana')); // true

// Removing values
mySet.delete('apple');

// Iterating through Set
for (let item of mySet) {
    console.log(item);
}
