Union-Find (Disjoint Set) Data Structure

Union-Find is a data structure that keeps track of elements partitioned into disjoint (non-overlapping) subsets.

It supports three main operations:

  • MakeSet(x): Creates a new set containing element x
  • Find(x): Returns the representative (root) of the set containing x
  • Union(x, y): Merges the sets containing x and y

This implementation uses path compression and union by rank for optimal performance.
