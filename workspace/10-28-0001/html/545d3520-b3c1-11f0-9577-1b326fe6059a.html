Huffman Coding

Huffman Coding is a popular method of data compression. It works by assigning variable-length codes to input characters, with shorter codes assigned to more frequent characters. The result is a prefix code that allows for efficient data storage and transmission.

How it Works

  1. Create a frequency table of the characters in the input.
  2. Build a binary tree using the frequencies.
  3. Assign binary codes to each character based on their position in the tree.

Example

For example, consider the input string "ABRACADABRA". The frequency of characters might look like this:

  • A: 5
  • B: 2
  • C: 1
  • D: 1
  • R: 2

The resulting Huffman codes might be:

  • A: 0
  • B: 101
  • C: 1110
  • D: 1111
  • R: 100

The compressed output for "ABRACADABRA" using Huffman coding might look like: "01010011100101001010100".
