What is the Two Pointers Technique?

The Two Pointers technique is a common algorithmic pattern used to solve problems involving arrays, strings, or linked lists. It involves using two pointers (indices or references) that traverse the data structure in a specific manner to find a solution more efficiently than a naive approach.

How it works:

// Example: Finding a pair with target sum in a sorted array
function twoSum(arr, target) {
  let left = 0;
  let right = arr.length - 1;

  while (left < right) {
    const sum = arr[left] + arr[right];

    if (sum === target) {
      return [left, right];
    } else if (sum < target) {
      left++; // Need a larger sum
    } else {
      right--; // Need a smaller sum
    }
  }

  return [-1, -1]; // No solution found
}
