An adjacency matrix is a square matrix used to represent a finite graph. The elements of the matrix indicate whether pairs of vertices are adjacent or not in the graph.

Adjacency Matrix

About Adjacency Matrices

Definition: An adjacency matrix is a way to represent which vertices (nodes) of a graph are adjacent to which other vertices.

Properties:

  • For an unweighted graph, matrix cells contain 0 or 1
  • For a weighted graph, cells contain weights or ∞ for no connection
  • Space complexity: O(V²) where V is the number of vertices
  • Checking if two vertices are connected: O(1) time complexity

Use Cases: Adjacency matrices are useful for dense graphs and when frequent connectivity checks are needed.
