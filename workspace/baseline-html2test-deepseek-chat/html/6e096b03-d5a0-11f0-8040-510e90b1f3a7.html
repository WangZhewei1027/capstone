Deque (Double-Ended Queue) Visualization

A deque (double-ended queue) is a data structure that allows insertion and deletion of elements from both ends. It combines the features of both stacks and queues, providing more flexibility in data manipulation.

How Deque Works:

Deque allows adding and removing elements from both ends. This makes it more versatile than a regular queue or stack.

// JavaScript implementation of a simple deque class Deque { constructor() { this.items = []; } // Add element to the front insertFront(element) { this.items.unshift(element); } // Add element to the rear insertRear(element) { this.items.push(element); } // Remove element from the front deleteFront() { if (this.isEmpty()) return "Underflow"; return this.items.shift(); } // Remove element from the rear deleteRear() { if (this.isEmpty()) return "Underflow"; return this.items.pop(); } // Check if deque is empty isEmpty() { return this.items.length === 0; } // Get front element getFront() { if (this.isEmpty()) return "No elements"; return this.items[0]; } // Get rear element getRear() { if (this.isEmpty()) return "No elements"; return this.items[this.items.length - 1]; } }
