Heap Sort Algorithm Explanation

Heap Sort is a comparison-based sorting algorithm that uses a binary heap data structure.

Steps:

  1. Build Max Heap: Convert the array into a max heap (parent nodes are greater than children)
  2. Extract Maximum: Repeatedly remove the maximum element from the heap and reconstruct the heap
  3. Sort: Place the extracted elements at the end of the array to form the sorted portion

Time Complexity:

  • Best Case: O(n log n)
  • Average Case: O(n log n)
  • Worst Case: O(n log n)

Space Complexity:

  • O(1) - sorts in place
