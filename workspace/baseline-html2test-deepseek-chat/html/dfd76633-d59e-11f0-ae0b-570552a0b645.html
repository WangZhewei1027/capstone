How Insertion Sort Works:

Insertion Sort builds the final sorted array one item at a time. It:

  1. Starts with the second element (considering the first element as sorted)
  2. Compares the current element with the elements before it
  3. Shifts larger elements to the right to make space
  4. Inserts the current element in its correct position
  5. Repeats until the entire array is sorted

Space Complexity: O(1)
