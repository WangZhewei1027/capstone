Deque Operations and Complexity

Deques provide O(1) time complexity for insertion and deletion at both ends when implemented with a doubly-linked list or dynamic array.

Operation Description Time Complexity
Push Front Add an element to the front O(1)
Push Back Add an element to the back O(1)
Pop Front Remove an element from the front O(1)
Pop Back Remove an element from the back O(1)
Get Front Get the front element without removing O(1)
Get Back Get the back element without removing O(1)
