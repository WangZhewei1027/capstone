How it works: This implementation uses the HTML5 Drag and Drop API along with JavaScript event handlers to enable reordering of list items.

Implementation Guide

Step 1: HTML Structure

Create a basic list structure with draggable items:

<ul id="sortable-list">
    <li draggable="true" data-id="1">Item 1</li>
    <li draggable="true" data-id="2">Item 2</li>
    <li draggable="true" data-id="3">Item 3</li>
</ul>

Step 2: CSS Styling

Add visual feedback for drag states:

li.dragging {
    opacity: 0.5;
}

li.over {
    border-top: 2px solid blue;
}

Step 3: JavaScript Logic

Set up event listeners for drag events:

const list = document.getElementById('sortable-list');
let draggedItem = null;

// Drag start event
list.addEventListener('dragstart', (e) => {
    draggedItem = e.target;
    e.target.classList.add('dragging');
});

// Drag over event
list.addEventListener('dragover', (e) => {
    e.preventDefault();
    const afterElement = getDragAfterElement(list, e.clientY);
    const currentItem = document.querySelector('.dragging');
    
    if (afterElement == null) {
        list.appendChild(currentItem);
    } else {
        list.insertBefore(currentItem, afterElement);
    }
});

// Drag end event
list.addEventListener('dragend', (e) => {
    e.target.classList.remove('dragging');
});

Complete Code

The full implementation includes helper functions for precise positioning:

function getDragAfterElement(container, y) {
    const draggableElements = [...container.querySelectorAll('li:not(.dragging)')];
    
    return draggableElements.reduce((closest, child) => {
        const box = child.getBoundingClientRect();
        const offset = y - box.top - box.height / 2;
        
        if (offset < 0 && offset > closest.offset) {
            return { offset: offset, element: child };
        } else {
            return closest;
        }
    }, { offset: Number.NEGATIVE_INFINITY }).element;
}

Pro Tip: You can extend this functionality by saving the new order to local storage or sending it to a server via AJAX.
