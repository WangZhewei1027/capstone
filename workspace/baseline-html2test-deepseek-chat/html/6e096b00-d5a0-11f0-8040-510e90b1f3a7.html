What is a Linked List?

A linked list is a linear data structure where each element (node) contains:

  • Data - The value stored in the node
  • Pointer/Reference - A link to the next node in the sequence

The last node points to NULL, indicating the end of the list.

Operations:

  • Add Node: Adds a new node with the specified value to the end of the list
  • Remove Node: Removes the first occurrence of the specified value
  • Find Node: Highlights the node containing the specified value
  • Traverse: Visits each node in sequence (animation)
  • Reverse: Reverses the order of nodes in the list
