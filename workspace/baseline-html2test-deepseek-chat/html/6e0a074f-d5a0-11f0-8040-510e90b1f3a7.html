Longest Common Subsequence

Find the longest sequence that appears in both strings in the same order

Input Strings

How it works:

The Longest Common Subsequence (LCS) algorithm finds the longest sequence of characters that appear in both input strings in the same order (but not necessarily consecutively).

Example: LCS of "ABCBDAB" and "BDCAB" is "BCAB" (length 4)
