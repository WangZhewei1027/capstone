About Heaps

Min Heap: A complete binary tree where each node is smaller than or equal to its children.

Max Heap: A complete binary tree where each node is larger than or equal to its children.

Operations: Insert O(log n), Extract Root O(log n), Get Min/Max O(1)

Common Uses: Priority queues, heap sort, Dijkstra's algorithm
