Adjacency List Visualization

An adjacency list is a collection of lists used to represent a finite graph. Each list describes the set of neighbors of a vertex in the graph. This data structure is efficient for sparse graphs and is widely used in graph algorithms.

Graph Input

Format: Each key is a vertex, and its value is an array of adjacent vertices.

Example: {"A": ["B", "C"], "B": ["A", "D"]} means A is connected to B and C, and B is connected to A and D.
