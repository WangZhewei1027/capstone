Depth-First Search (DFS)

Depth-First Search is a graph traversal algorithm that explores as far as possible along each branch before backtracking. It can be implemented recursively or iteratively with an explicit stack.

Key ideas

  • Use a visited set to avoid revisiting nodes.
  • Recursive DFS uses the call stack; iterative DFS uses a manual stack.
  • Neighbor order affects traversal order, but not correctness.
  • Works on trees and general graphs (directed or undirected).

Pseudocode

Recursive DFS

// DFS(G, start):
visited = empty set

function visit(v):
  if v in visited: return
  add v to visited
  for each n in G[v]:
    visit(n)

visit(start)

Iterative DFS (with a stack)

// DFS(G, start):
visited = empty set
stack = [start]

while stack not empty:
  v = stack.pop()
  if v not in visited:
    add v to visited
    // push neighbors so the first neighbor is visited first
    for each n in reverse(G[v]):
      if n not in visited:
        stack.push(n)
