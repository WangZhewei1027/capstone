Longest Common Subsequence (LCS) Calculator

What is Longest Common Subsequence?

The Longest Common Subsequence (LCS) of two strings is the longest sequence of characters that appear left-to-right (but not necessarily contiguously) in both strings.

For example, for the strings ABCBDAB and BDCABA, one longest common subsequence is BCBA.

This demo uses dynamic programming to build a matrix that tracks the length of LCS for all prefixes of the two strings, and then backtracks to find the actual subsequence.
