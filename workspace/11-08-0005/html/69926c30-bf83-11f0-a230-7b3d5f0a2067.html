Counting Sort is a linear time sorting algorithm that works by counting the occurrences of each element and using arithmetic to determine their positions in the output array.

How Counting Sort Works:

  1. Count the frequency of each element (0-9 in this demo)
  2. Calculate cumulative counts to determine positions
  3. Place each element in its correct position based on counts
  4. Time Complexity: O(n + k) where k is the range of input
