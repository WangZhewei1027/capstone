I'll design an interactive HTML page that provides an in-depth, hands-on exploration of the Floyd-Warshall Algorithm. ## Interactive Design Plan ### Concept Title **Floyd-Warshall Algorithm: Finding All Shortest Paths** ### Learning Objective Users will understand how the Floyd-Warshall algorithm systematically finds the shortest paths between all pairs of vertices in a weighted graph by considering intermediate vertices one at a time, visualizing the dynamic programming process through interactive step-by-step execution. ### Interaction Design The page features: 1. **Graph Visualization Canvas**: A interactive graph where users can: - Click to add vertices (nodes) - Click and drag between vertices to create edges - Click on edges to set weights - Delete vertices/edges with right-click 2. **Algorithm Controls**: - "Run Algorithm" button to start the process - "Step Forward/Backward" buttons for manual progression - "Reset" button to clear the algorithm state - Speed slider to control animation pace 3. **Visual Feedback**: - Currently considered intermediate vertex highlights in red - Source and destination vertices pulse with different colors - Path being evaluated shows with animated dashed lines - Distance matrix cells animate when values update - Shortest path highlights in green when found 4. **Distance Matrix Display**: - Live-updating matrix showing distances between all vertex pairs - Cells highlight when being compared or updated - Infinity symbol (∞) for unreachable paths ### Layout Description - **Top Section** (24px from viewport top): Title and brief algorithm description - **Main Content Area** (centered, max-width 1200px): - **Left Panel** (60% width): Graph canvas with vertex/edge creation tools - **Right Panel** (40% width): Distance matrix and algorithm state - **Control Bar** (bottom, full width): Algorithm controls and speed slider - **Info Panel** (collapsible): Detailed step explanations and pseudocode All interactive elements maintain 16px minimum spacing, with 24px safe margins from viewport edges.
