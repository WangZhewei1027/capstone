How Heap Sort Works

Heap Sort uses a binary heap data structure to sort elements. First, it builds a max-heap where each parent is greater than its children. Then, it repeatedly extracts the maximum element (root) and places it at the end of the array. Time Complexity: O(n log n) | Space Complexity: O(1)
