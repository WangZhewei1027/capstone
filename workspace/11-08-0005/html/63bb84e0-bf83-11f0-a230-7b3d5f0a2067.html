How Linear Search Works

Linear search is the simplest searching algorithm that checks each element in sequence until it finds the target or reaches the end.

Time Complexity: O(n) - In the worst case, it checks every element

Space Complexity: O(1) - Only uses a constant amount of extra space

Best for: Small datasets, unsorted arrays, or when simplicity is preferred over efficiency
