Set Properties

  • ✓ Only unique values
  • ✓ Any type of value

Array vs Set

// Array allows duplicates
[1, 2, 2, 3] // [1, 2, 2, 3]

// Set removes duplicates
new Set([1, 2, 2, 3]) // {1, 2, 3}
  • Arrays: Ordered, indexed, duplicates OK
  • Sets: Unique values, no index access
  • Convert: [...mySet] or Array.from(mySet)
