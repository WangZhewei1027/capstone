DFS

This is a simple example of DFS using JavaScript. The function "find" takes two arguments: an array and a target node. The function will recursively visit all nodes in the array, starting from the target node, until it reaches a leaf node or returns null. ```javascript function find(arr, target) { if (arr === null || arr.length === 0) return null; for (let i = 0; i < arr.length - 1; i++) { if (arr[i] === target) return i; } return null; } ``` In this example, we define a function called `find` that takes two parameters: an array and a target node. We check if the array is null or empty before trying to access any element. If it is, we return `null`. The function then iterates through the rest of the array, starting at the last element. For each element, we compare it to the target node. If they are equal, we return the index of the element. If the function has reached the end of the array without finding the target node, it returns `null`. When you call this function with an array and a target node, it will return the index of the first element that matches the target node. For example, let's say we have an array `[1,2,3,4,5]`, a target node `target` of `5`. When we call `find([1,2,3,4,5], 5)`: ```javascript console.log(find([1,2,3,4,5], 5)); // Output: 2 ``` This means that the function找到了 the index of the element `5` in the array `[1,2,3,4,5]`.
