Priority Queue

Priority Queue is a data structure that stores elements in a specific order, where the first element is always at the front of the queue and the last element is always at the back of the queue.

Size

The size of a priority queue can be calculated using the following formula:

Size = Number of elements + Number of positions - 1

Queue Operations

  • Pop: Removes the topmost element from the queue and returns it to its original position.
  • Enqueue: Adds an element to the front of the queue and returns it to its original position.
  • Dequeue: Removes the backmost element from the queue and returns it to its original position.
  • Check: Returns true if the queue is empty, false otherwise.
