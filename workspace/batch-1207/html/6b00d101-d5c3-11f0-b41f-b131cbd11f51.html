Linear Search Algorithm Demo

Linear Search Algorithm Code


function linearSearch(arr, target) {
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] === target) {
            return i; // Return index if found
        }
    }
    return -1; // Return -1 if not found
}

Algorithm Explanation

Linear Search is a simple searching algorithm that:

  • Starts from the first element of the array
  • Compares each element with the target value
  • Returns the index if a match is found
  • Continues until the end of the array if no match is found
  • Returns -1 if the target is not in the array

Time Complexity: O(n) - In the worst case, it checks every element

Space Complexity: O(1) - It uses a constant amount of extra space
