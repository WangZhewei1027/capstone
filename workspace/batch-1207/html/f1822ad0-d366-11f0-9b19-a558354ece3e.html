How LCS Works:

The Longest Common Subsequence algorithm uses dynamic programming to find the longest sequence that appears in the same order in both sequences (but not necessarily consecutively).

Algorithm:

  1. Create a table with dimensions (m+1) Ã— (n+1) where m and n are the lengths of the sequences
  2. Initialize the first row and column to 0
  3. For each cell (i,j):
    • If characters match: table[i][j] = table[i-1][j-1] + 1
    • Otherwise: table[i][j] = max(table[i-1][j], table[i][j-1])
  4. The bottom-right cell contains the length of the LCS
  5. Trace back through the table to find the actual LCS
