The outer loop iterates from the first element to the second-to-last element in the array, while the inner loop iterates from the second-to-last element to the last element in the array. If any element is found that is greater than its adjacent element, it is swapped. After all elements have been processed, the function returns the sorted array. You can call this function with your own array as an argument to sort it.
