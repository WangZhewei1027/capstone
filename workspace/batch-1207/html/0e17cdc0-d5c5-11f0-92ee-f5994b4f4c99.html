Here is an example of how to use Merge Sort in JavaScript: ``` function mergeSort(arr) { if (arr.length <= 1) return arr; let left = arr.slice(0, Math.min(arr.length, 2)); let right = arr.slice(Math.max(arr.length, 2)); let mid = Math.floor((left.length + right.length) / 2); merge(left, mid, right); return merge(left, mid, right); } function merge(left, mid, right) { while (mid > 0 && left[mid - 1] > right[mid]) { left[mid] = left[mid - 1]; mid--; } while (mid < 0 && left[mid + 1] > right[mid]) { left[mid + 1] = left[mid + 1]; mid++; } while (right < left.length && right[mid + 1] > mid) { right[mid + 1] = right[mid + 1]; mid++; } if (mid === 0) return null; let temp = left[mid]; left[mid] = right[mid]; right[mid] = temp; return arr.concat(left, middle - 1, right); } ``` This code defines a `mergeSort` function that takes an array as input and returns a sorted array using Merge Sort. The function first checks whether the array has at least two elements or not, and then divides it into three subarrays: one containing all the elements less than the middle element, one containing all the elements equal to the middle element, and one containing all the elements greater than the middle element. It then recursively calls itself on each subarray and concatenates them back together to form the final sorted array. To use this function, you can call it like this: ``` let arr = [3, 1, 4, 6, 8, 5]; console.log(mergeSort(arr)); // Output: [1, 2, 3, 4, 5, 6] ```
