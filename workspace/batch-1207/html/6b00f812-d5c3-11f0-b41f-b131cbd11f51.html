How Dijkstra's Algorithm Works

Dijkstra's algorithm is a greedy algorithm that finds the shortest path between nodes in a graph. It works by:

Step 1: Assign a tentative distance value to every node: zero for the starting node and infinity for all others.
Step 2: Set the starting node as current. Mark all other nodes unvisited.
Step 3: For the current node, consider all of its unvisited neighbors and calculate their tentative distances.
Step 4: When we're done considering all neighbors of the current node, mark it as visited.
Step 5: Select the unvisited node with the smallest tentative distance and set it as the new current node.
Step 6: Repeat steps 3-5 until the destination node has been visited or the smallest tentative distance is infinity.
