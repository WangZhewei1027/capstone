Insertion Sort Algorithm

How Insertion Sort Works:

Insertion sort builds the final sorted array one item at a time. It iterates through the input array and at each iteration, it removes one element from the input data, finds the location it belongs within the sorted list, and inserts it there.

Time Complexity: O(nÂ²) in worst case, O(n) in best case

Space Complexity: O(1)

function insertionSort(arr) {
    for (let i = 1; i < arr.length; i++) {
        let key = arr[i];
        let j = i - 1;
        
        // Move elements that are greater than key
        // one position ahead of their current position
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j = j - 1;
        }
        arr[j + 1] = key;
    }
    return arr;
}
