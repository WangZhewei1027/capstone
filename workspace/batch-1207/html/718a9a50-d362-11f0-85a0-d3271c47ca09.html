Breadth-First Search Example

This is an example of BFS to search for a given element in a tree.

The input tree will be represented by a list of nodes and their values, where each node represents a level in the tree.

Example Code:

`const tree = [ { value: 'A', left: { children: [ { value: 'B' }, { value: 'C' } ] }, { value: 'D', left: { children: [ { value: 'E' }, { value: 'F' } ] }, { value: 'G', left: { children: [ { value: 'H' }, { value: 'I' } ] }, { value: 'J', left: { children: [ { value: 'K' }, { value: 'L' } ] } ];`

`const node = tree.find(node => node.value === 'A');`

`node.left.forEach(child => child.left);`

`node.right.forEach(child => child.right);`

`console.log(node.value);` // Output: 'A'
