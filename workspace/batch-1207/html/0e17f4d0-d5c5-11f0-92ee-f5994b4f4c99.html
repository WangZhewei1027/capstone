Sure, here's an example of a quicksort algorithm implemented in JavaScript: ```javascript function quickSort(arr) { if (arr.length <= 1) return arr; pivot = arr[Math.floor((arr.length / 2) * Math.random() + 1)]; left = [x for x in arr if x < pivot]; middle = [x for x in arr if x == pivot]; right = [x for x in arr if x > pivot]; return quickSort(left) + middle + quickSort(right); } ``` This function takes an array as input and recursively sorts it by partitioning it into three sub-arrays based on a random pivot element. The `Math.floor` function is used to ensure that each sub-array has an equal size.
