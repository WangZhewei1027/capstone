What is a Hash Map?

A hash map (or hash table) is a data structure that implements an associative array, allowing you to store key-value pairs.

How it works:

  • Hashing: A hash function converts the key into an index (hash code)
  • Buckets: The hash map is divided into buckets (or slots)
  • Collision Handling: When multiple keys hash to the same index, collisions are handled (usually with chaining)
  • Operations: Constant-time O(1) average case for insert, delete, and search operations

Key Operations

set(key, value): Adds a key-value pair

get(key): Retrieves the value for a key

has(key): Checks if a key exists

delete(key): Removes a key-value pair

Characteristics

Fast Access: O(1) average time complexity

Dynamic Sizing: Can grow as needed

No Duplicates: Each key is unique

Collision Resistant: Handles key collisions

In this implementation, we use a simple hash function and handle collisions by chaining (linked lists in each bucket).
