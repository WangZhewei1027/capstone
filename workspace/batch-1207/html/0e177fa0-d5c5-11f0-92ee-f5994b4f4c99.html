Selection Sort

The Selection Sort algorithm works by repeatedly selecting an element from unsorted elements and placing it in its correct position within the sorted array.

  1. Start at the beginning of the array and work your way to the end.
  2. Move the first unsorted element to the end of the array.
  3. Move the second unsorted element to the left of the first unsorted element.
  4. Repeat steps 2-4 until all elements have been moved.
