What is Recursion?

Recursion is a method of solving problems where the solution depends on solutions to smaller instances of the same problem. A recursive function typically has two parts:

  • Base Case: The condition that stops the recursion
  • Recursive Case: The part where the function calls itself with a modified argument
function factorial(n) { // Base case if (n <= 1) return 1; // Recursive case return n * factorial(n - 1); }

Without a proper base case, a recursive function would call itself indefinitely, causing a stack overflow.
