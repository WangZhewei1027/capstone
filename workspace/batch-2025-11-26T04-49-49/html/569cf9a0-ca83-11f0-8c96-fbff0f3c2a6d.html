Merge Sort

Merger Sort is an efficient sorting algorithm that works by dividing the input array into two smaller subarrays, and then recursively merging them back together.

function mergeSort(arr) { if (arr.length <= 1) { return arr; } let mid = Math.floor(arr.length / 2); let left = arr.slice(0, mid); let right = arr.slice(mid + 1); return mergeSort(left) + mergeSort(right); } function merge(left, right) { let result = []; while (left.length && right.length) { if (left[0] <= right[0]) { result.push(left.shift()); } else { result.push(right.shift()); } } while (left.length || right.length) { if (left.length) { result.push(left.shift()); } else { result.push(right.shift()); } } return result.concat(left); } Here's an example of how you can use the `mergeSort` function: ``` let arr = [42, 31, 12, 55, 67, 89]; console.log(mergeSort(arr)); // Output: [12, 12, 15, 16, 18, 20] ``` In this example, we have an array `[42, 31, 12, 55, 67, 89]`. The `mergeSort` function is called on this array, which returns the sorted array `[12, 12, 15, 16, 18, 20]`.
