Kruskal's Algorithm

Prerequisites:

  • Given a list of edges, where each edge is represented by a tuple (u, v, w)
  • The graph is represented as an adjacency list

Input:

        
            let edges = [
                ["A", "B", 2],
                ["B", "C", 3],
                ["A", "C", 4],
                ["C", "D", 5],
                ["D", "E", 6]
            ];

Output:

        
            let graph = {
                "A": ["B", "C"],
                "B": ["A", "C"],
                "C": ["A", "B", "D"],
                "D": ["C", "E"],
                "E": ["D"]
            };
            let sortedEdges = kruskal(edges, graph);
            console.log(sortedEdges);
